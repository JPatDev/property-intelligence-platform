# Property Intelligence Requirements

## Purpose

Provide claim professionals with comprehensive property data and event history.

## Property Profile

Store:

* Address
* Parcel ID
* County
* Year Built
* Roof Type
* Square Footage
* Ownership Information

## Permit Intelligence

Track:

* Permit History
* Roof Permits
* Repair Permits
* Contractor Information

## Weather Intelligence

Track:

* Hail Events
* Wind Events
* Hurricanes
* Tornadoes

## Historical Imagery

Store references to:

* Aerial Images
* Satellite Images
* Inspection Photos

## Property Timeline

Generate chronological history:

* Construction
* Permits
* Storm Events
* Inspections
* Claims

## Property Risk Score

Calculate risk based on:

* Property Age
* Roof Age
* Weather Exposure
* Prior Claims
* Permit Activity
