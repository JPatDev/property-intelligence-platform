# Azure SaaS Architecture

## Purpose
Defines the Azure deployment architecture for ClaimsOS.

## Core Services
- Azure App Service or Azure Container Apps
- Azure Database for PostgreSQL Flexible Server
- Azure Blob Storage
- Azure Key Vault
- Azure OpenAI
- Azure AI Document Intelligence
- Azure Service Bus
- Azure Monitor
- Application Insights
- Azure Front Door

## Deployment Strategy
- Dev
- Test
- Staging
- Production

## Non-Functional Requirements
- High Availability
- Disaster Recovery
- Auto Scaling
- Security by Default
